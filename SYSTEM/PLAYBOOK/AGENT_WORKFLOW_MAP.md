# AGENT WORKFLOW MAP

# 1. PROFIT AUDITOR

Purpose:
Calculate true net profit across products, bundles, categories, collections, goals, and ads.

Responsibilities:
- Calculate daily profit
- Track profit velocity
- Track profit stability
- Track refund risk
- Feed product tier rankings
- Feed Top 30 rankings
- Feed capital allocator

Outputs:
- Daily profit logs
- SKU profitability ranking
- Weighted Profit Score inputs

---

# 2. PRODUCT TIER MANAGER

Purpose:
Rank and manage all products using the Tier system.

Responsibilities:
- Weekly re-ranking
- Daily monitoring
- Tier movement alerts
- Tier 4 recovery tracking
- Enforce 2-month Tier 4 cutoff

Outputs:
- Tier assignments
- Promotion candidates
- Pause recommendations
- Recovery alerts

---

# 3. CAPITAL ALLOCATOR

Purpose:
Distribute marketing funds using profit-first rules.

Responsibilities:
- Allocate 80% budget to Top 30 ads
- Allocate 20% to Promotion Pool
- Reallocate failed Tier 4 spend
- Detect scalable winners
- Protect Tier 1 products

Outputs:
- Daily budget allocations
- Spend increase recommendations
- Spend decrease recommendations
- Promotion Pool rankings

---

# 4. CREATIVE INTELLIGENCE AGENT

Purpose:
Scrape and deconstruct viral ads for strategic learning.

Responsibilities:
- Discover viral ads
- Analyze WHY ads worked
- Extract hooks
- Extract pacing
- Extract CTAs
- Extract emotional triggers
- Extract visual patterns
- Generate audience hypotheses
- Suggest recreation strategies

Outputs:
- Viral deconstruction reports
- Recreation prompts
- Product fit suggestions
- Creative delta recommendations

---

# 5. EXPERIMENT GOVERNOR

Purpose:
Control A/B testing and experimentation.

Responsibilities:
- Run Baseline A tests
- Run Challenger B tests
- Decide test duration
- Kill weak performers
- Promote winners
- Archive learnings

Outputs:
- Test results
- Winning formats
- Archived experiments
- Promotion recommendations

---

# 6. LIBRARY STEWARD

Purpose:
Manage creative libraries and vault systems.

Responsibilities:
- Keep Top 5 creatives per SKU
- Maintain Fallout library
- Maintain Elite Top 20
- Remove weak creatives
- Preserve historical winners

Outputs:
- Active creative library
- Fallout archive
- Elite vault
- Best-of-best render folder

---

# 7. EXECUTIVE REPORTER

Purpose:
Generate operational reports.

Responsibilities:
- Daily reporting
- Weekly reporting
- Monthly reporting
- Quarterly analysis
- Long-term pattern detection

Outputs:
- Profit reports
- Tier movement reports
- Capital allocation reports
- Awareness and momentum reports
- Executive summaries
